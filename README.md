# 🔢 Sorting Visualizer (DSA)

A modern and interactive **Sorting Algorithm Visualizer** built using **HTML, CSS, and JavaScript**.

## 🚀 Live Demo
👉 https://your-username.github.io/sorting-visualizer/

---

## ✨ Features

- 🎯 Visualize multiple sorting algorithms
- 🔊 Sound effects based on values
- 🎨 Color-coded animations:
  - 🟡 Comparing
  - 🔴 Swapping
  - 🟢 Sorted
- ⚡ Adjustable speed control
- 🎲 Random array generation (values 1–100)
- 📊 Clean bar chart visualization

---

## 🧠 Algorithms Implemented

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
